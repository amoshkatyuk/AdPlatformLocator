## Описание

Веб-сервис для поиска рекламных площадок по регионам.  
Площадки загружаются из файла, данные хранятся в памяти.

---

## Запуск проекта

*Ниже представлены команды для ввода в консоль Windows

### 1. Клонирование репозитория

git clone https://github.com/amoshkatyuk/AdPlatformLocator
cd AdPlatformLocator
cd AdPlatformLocator.API

### 2. Проверка, что установлен .NET 8 SDK

dotnet --version

### 3. Запуск веб-сервиса

dotnet run

После запуска в консоли появится адрес, например:
Now listening on: http://localhost:5222

### 4. Переход в Swagger UI

Открыть в браузере http://localhost:5222/swagger
